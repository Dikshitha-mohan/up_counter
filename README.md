# 4-bit Up Counter using Verilog HDL

## Project Overview

This project demonstrates the design and simulation of a 4-bit synchronous up counter using Verilog HDL.

The counter increments by one on every positive edge of the clock and resets to zero when the reset signal is asserted.

## Features

- Verilog RTL Design
- Asynchronous Reset
- Positive Edge Triggered
- Testbench Included
- Simulated using EDA Playground
- Waveform verified in EPWave

## Tools

- Verilog HDL
- EDA Playground
- Icarus Verilog
- EPWave

## Files

- counter.v
- counter_tb.v
- waveform.png

## Simulation Output

The waveform verifies:

- Clock generation
- Reset functionality
- Counter increments correctly
